# 实现逻辑

  • 在 layout 组件中检测子组件是否含有sider 组件
  • 如果为 true 那么 flex-direction：row
  • 如果为false那么为 column 方式排列
mounted(){
    this.$children.forEach((vm)=>{
      if (vm.$options.name === 'Sider') {
        this.layoutClass.hasSider = true
      }
    })
  }
